# Coffee Feedback Frontend

Vue 3 + Vite + Tailwind CSS frontend for the Coffee Tester Feedback Collection system.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:8000
```

## Features

- Vue 3 Composition API
- Pinia for state management
- Vue Router for navigation
- Tailwind CSS for styling
- Web Speech API integration
- Real-time voice transcription
- Animated UI components
- Responsive design

## Browser Support

- Chrome 25+ (recommended)
- Edge 79+
- Safari 14.1+
- Firefox (partial voice support)

Voice features require Chrome, Edge, or Safari for full functionality.
